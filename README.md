# Cookbook

Collection of runnable AI pipelines.

## Available Pipeline

- [`multimodal-rag-nemotron`](multimodal-rag-nemotron): Generic multimodal RAG for PDF/Markdown documents with local models and Chainlit UI.

## Repository Layout

- Each pipeline lives under a dedicated top-level folder.
- Each pipeline keeps its own `README.md`, `pyproject.toml`, and `uv.lock`.
- Run commands from the pipeline folder unless stated otherwise.
